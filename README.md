# Incident-In-The-Woods-Game

A small 2D pixel style runner made in Unity.
You play as a witch-deer creature running through the forest, dodging obstacles and trying to get the highest score.
itch.io: https://wloexpel.itch.io/incidentinthewoods

### How to Play
Open scene:
Assets/Scenes/MainScene.unity
Press space once or twice to avoid obstacles
Try to beat your high score
When game over — press Try Again button to start over

### How to Open the Project
1. Install Unity 2022.3.50f1
2. Clone the project https://github.com/wloexPel/Incident-In-The-Woods-Game
3. Open Unity Hub
4. Click Add project
5. Select this folder

### How to Build the Game
1. Go to File → Build Settings.
2. Select PC, Mac & Linux Standalone
3. Choose Windows
4. Press Build
